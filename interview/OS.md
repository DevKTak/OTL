```
Q. List vs Set vs Map
A.
- List는 기본적으로 데이터들이 순서대로 저장되며 중복을 허용한다.
- Set은 순서가 보장되지 않고 데이터들의 중복을 허용하지 않는다.
- Map은 순서가 보장되지 않고 Key 값의 중복은 허용하지 않지만 Value 값의 중복은 허용된다.
```
